# Glem Website

Base website in Next.js, ready for GitHub + Vercel deployment.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Content folders

- `01.Home`: add a cover video (`.mp4`, `.webm`, `.mov`)
- `02.About us`: add team images and optional text file (`.txt`/`.md`)
- `03.Project/<project-name>/COP`: first image is used as project cover
- `03.Project/<project-name>/DEV`: all images are shown inside project modal
- `04.Events`: add event images and optional text files
- `05.Contacts`: add a text file with contact details

## Pages

- `/it` and `/en` (language switch available in header)
- Home, About, Projects, Events, Contacts
